The Three Theories of Truth

Not all truths are tested the same way. Philosophers use three main "tools" to determine if a statement is true.

1. Correspondence

The "Reality Check"

Definition:

A statement is true if it corresponds (matches) to the way the world actually is.

The Test:

Observation / Senses

Example: "The cat is on the mat." (True only if you look and see the cat on the mat).

2. Coherence

The "System Check"

Definition:

A statement is true if it coheres (fits logically) with a system of other established beliefs or rules.

The Test:

Logic / Consistency

Example: "2 + 2 = 4" or "A bachelor is an unmarried male." (True because it fits the rules of Math/English).

3. Pragmatic

The "Utility Check"

Definition:

A statement is true if holding it to be true works useful or beneficial in practice.

The Test:

Results / Application

Example: "Democracy is the best government." (True if it results in a better society than alternatives).

Lesson Summary

  • • Correspondence: Best for physical facts (Is it raining?).
  • • Coherence: Best for abstract systems like Math, Logic, or Law (Does it fit the rules?).
  • • Pragmatic: Best for complex social/moral ideas where results matter (Does it work?).
